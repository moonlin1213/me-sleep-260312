# 疗愈网页部署指南

## 📍 你的情况

- **前端网页**：`~/Desktop/ME-Sleep-260216-01`（React + Vite）
- **后端服务器**：`~/Desktop/mem-260223/server`（NestJS）
- **部署方式**：腾讯云静态网页托管

---

## 🔍 第一步：确认后端地址

### 问题：你的后端服务器在哪里？

**选项 A：后端在本地运行**
- 地址：`http://localhost:3000`
- ❌ **不能用于生产环境**（只能在你的电脑上访问）

**选项 B：后端在腾讯云服务器**
- 地址：`https://你的域名` 或 `http://你的服务器IP:3000`
- ✅ **可以用于生产环境**

**选项 C：后端在腾讯云云函数**
- 地址：`https://service-xxx.gz.apigw.tencentcs.com`
- ✅ **可以用于生产环境**

---

### 如何确认？

**方法 1：检查小程序的 API 配置**

打开 `~/Desktop/mem-260223/src/network.ts`，看看 `API_BASE_URL` 是什么：

```typescript
const API_BASE_URL = 'http://172.20.10.6:3000' // 这是你的后端地址
```

**当前配置**：`http://172.20.10.6:3000`
- 这是一个**局域网地址**（你的电脑在局域网的 IP）
- ❌ **不能用于生产环境**（只能在同一 WiFi 下访问）

---

### 你需要做什么？

**如果后端还没有部署到云服务器**，你有两个选择：

#### 选择 1：部署后端到腾讯云服务器（推荐）

**步骤**：
1. 购买腾讯云服务器（轻量应用服务器即可，最便宜的就够用）
2. 在服务器上安装 Node.js
3. 上传后端代码到服务器
4. 运行后端服务
5. 配置域名（可选，但推荐）

**优点**：
- 稳定，24 小时运行
- 可以配置域名
- 性能好

**缺点**：
- 需要购买服务器（约 50-100 元/月）
- 需要一些运维知识

#### 选择 2：部署后端到腾讯云云函数（免费）

**步骤**：
1. 在腾讯云云开发控制台创建云函数
2. 上传后端代码
3. 配置触发器（HTTP 触发）
4. 获取云函数的 URL

**优点**：
- 免费（每月有免费额度）
- 不需要运维
- 自动扩容

**缺点**：
- 冷启动慢（第一次访问需要几秒）
- 有一些限制（比如超时时间）

---

## 🚀 第二步：修改 API URL

### 假设你的后端地址是：`https://api.memjoy.com`

**1. 打开 `.env.local` 文件**

```bash
cd ~/Desktop/ME-Sleep-260216-01
open .env.local
```

**2. 修改 `REACT_APP_API_URL`**

```
GEMINI_API_KEY=PLACEHOLDER_API_KEY
REACT_APP_API_URL=https://api.memjoy.com
```

**保存文件。**

---

## 🔨 第三步：构建前端

### 1. 安装依赖（如果还没安装）

```bash
cd ~/Desktop/ME-Sleep-260216-01
npm install
```

### 2. 构建生产版本

```bash
npm run build
```

**会发生什么？**
- Vite 会把你的 React 代码打包成静态文件
- 生成一个 `dist` 目录
- `dist` 目录里包含：
  - `index.html`（入口文件）
  - `assets/`（CSS、JS、图片等）

**构建完成后，你会看到**：

```
✓ built in 5.23s
dist/index.html                   0.45 kB
dist/assets/index-abc123.js      123.45 kB
dist/assets/index-def456.css      12.34 kB
```

## 📦 第四步：部署到腾讯云静态网页托管

### 方法 1：通过腾讯云控制台（推荐，最简单）

**1. 打开腾讯云控制台**

访问：https://console.cloud.tencent.com/tcb

**2. 进入你的环境**

- 点击你的云开发环境（应该已经创建过了）
- 左侧菜单 → 静态网站托管

**3. 上传文件**

- 点击「文件管理」
- 点击「上传文件」
- **选择 `dist` 目录里的所有文件**（不是 `dist` 目录本身）
- 上传

**4. 等待上传完成**

- 上传完成后，你会看到文件列表
- 包括 `index.html`、`assets/` 等

**5. 访问你的网站**

- 点击「设置」
- 找到「默认域名」
- 复制域名，访问

---

### 方法 2：通过命令行（高级）

**1. 安装腾讯云 CLI**

```bash
npm install -g @cloudbase/cli
```

**2. 登录**

```bash
tcb login
```

**3. 部署**

```bash
cd ~/Desktop/ME-Sleep-260216-01
tcb hosting deploy dist -e 你的环境ID
```

**环境 ID 在哪里？**
- 打开腾讯云控制台
- 云开发 → 环境 → 环境 ID

## ⚠️ 常见问题

### 问题 1：构建失败

**错误信息**：
```
Error: Cannot find module 'xxx'
```

**解决方法**：
```bash
npm install
npm run build
```

---

### 问题 2：上传后访问 404

**原因**：
- 上传的文件路径不对
- 应该上传 `dist` 目录里的文件，不是 `dist` 目录本身

**解决方法**：
1. 删除腾讯云上的所有文件
2. 重新上传 `dist` 目录里的文件

---

### 问题 3：API 请求失败

**错误信息**（浏览器控制台）：
```
Failed to fetch
```

**原因**：
- API URL 配置错误
- 后端服务器没有运行
- CORS 配置错误

**解决方法**：
1. 检查 `.env.local` 里的 `REACT_APP_API_URL`
2. 确认后端服务器正在运行
3. 检查后端的 CORS 配置

---

### 问题 4：设备指纹验证失败

**错误信息**：
```
设备指纹不匹配
```

**原因**：
- 后端没有运行
- 后端的 `/api/verify-device` 接口有问题

**解决方法**：
1. 检查后端是否运行
2. 访问 `https://你的后端地址/api/verify-device`，看是否返回 404
