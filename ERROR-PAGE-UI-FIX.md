## 🎨 呼吸光晕效果

按钮使用了 `btn-main-glow` class，这个 class 在 `index.html` 的 `<style>` 标签里定义：

```css
@keyframes main-btn-glow {
  0% { 
    box-shadow: 0 0 5px rgba(212, 175, 55, 0.1); 
    border-color: rgba(212, 175, 55, 0.2); 
  }
  50% { 
    box-shadow: 0 0 20px rgba(212, 175, 55, 0.3); 
    border-color: rgba(212, 175, 55, 0.6); 
    transform: scale(1.01); 
  }
  100% { 
    box-shadow: 0 0 5px rgba(212, 175, 55, 0.1); 
    border-color: rgba(212, 175, 55, 0.2); 
  }
}

.btn-main-glow {
  animation: main-btn-glow 6s infinite ease-in-out;
}
```

**效果**：
- 6 秒一个周期
- 缓慢呼吸（阴影和边框颜色变化）
- 轻微缩放（1.01 倍）

## ⚠️ 注意事项

### 1. 字体

确保用户设备上有 `Noto Sans SC` 字体，否则会回退到系统默认字体。

### 2. 呼吸光晕效果

呼吸光晕效果是 CSS 动画，所有现代浏览器都支持。

### 3. 粒子背景

粒子背景是 Canvas 动画，需要浏览器支持 Canvas API。
